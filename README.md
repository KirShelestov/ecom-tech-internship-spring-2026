## Бэкенд часть

В ходе тестового задания было реализовано моковое API.

1. перейти в /server
2. npm install
3. указать в .env через переменную PORT на каком порту поднимать сервер, иначе по умолчанию будет стартовать на 3333.
4. npm run dev

### Примеры запросов

GET http://localhost:3333/products - получить все товары
GET http://localhost:3333/products?search=смарт - поиск товара реализован на бэке
GET http://localhost:3333/products/1 - получить товар по id
